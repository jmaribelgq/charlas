CÓMO USARLO EN WORDPRESS — 3 MÉTODOS
═══════════════════════════════════════════════════════════════

MÉTODO 1: PÁGINA COMPLETA HTML (más sencillo)
─────────────────────────────────────────────
1. En WordPress, ve a Páginas → Añadir nueva
2. Cambia el editor a "Código" (botón < > en la esquina)
3. Pega TODO el contenido del archivo index.html
4. Publica la página
5. ✅ Listo — la guía completa aparecerá en esa URL

MÉTODO 2: BLOQUE HTML EN GUTENBERG
────────────────────────────────────
1. Crea una nueva página en WordPress
2. En el editor Gutenberg, agrega un bloque "HTML personalizado"
3. Copia el contenido entre <body> y </body> del index.html
4. El CSS va en Apariencia → Personalizar → CSS adicional
5. Copia el contenido entre <style> y </style> ahí

MÉTODO 3: TEMA HIJO CON PÁGINA COMPLETA
─────────────────────────────────────────
1. Sube el index.html como template de página
2. En tu tema hijo, crea: page-chicago.php
3. Llama al HTML con get_template_part()
4. Asigna ese template a tu página de Chicago

═══════════════════════════════════════════════════════════════
GOOGLE SITES (alternativa gratuita)
═══════════════════════════════════════════════════════════════
1. Ve a sites.google.com
2. Crea un sitio nuevo
3. Insertar → Incrustar → "Incrustar código"
4. Pega el HTML completo
5. ✅ Publicar y listo — URL pública gratuita
