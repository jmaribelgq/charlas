## 🌐 Cómo subir esta página a internet

### OPCIÓN 1: Subirla a tu sitio web actual

Si ya tienes un dominio (ejemplo: maribelgodoy.com o neuroeduca.com.gt):

1. Conéctate a tu hosting (cPanel, FileZilla, o el panel que uses)
2. Crea una carpeta nueva, por ejemplo: `/poder-invisible-mama/`
3. Sube TODOS los archivos del ZIP a esa carpeta
4. Tu página estará disponible en: `tudominio.com/poder-invisible-mama/`

### OPCIÓN 2: Si no tienes sitio web aún (GRATIS)

Puedes subirla gratis a estos servicios:

**Netlify (recomendado, súper fácil):**
1. Entra a https://app.netlify.com/drop
2. Arrastra TODA la carpeta descomprimida a la página
3. Listo, te dan un link como: `lindo-nombre-12345.netlify.app`
4. Puedes conectar tu propio dominio después si quieres

**GitHub Pages:** (requiere cuenta de GitHub)
**Vercel:** Similar a Netlify, también gratis

### OPCIÓN 3: WordPress

Si tu sitio es WordPress:
1. Instala el plugin "Insert HTML Snippet" o "WPCode"
2. Crea una página nueva
3. Pega el código HTML completo en un bloque "HTML personalizado"
4. Sube las imágenes a la biblioteca de medios
5. Cambia las rutas de las imágenes en el código

## 🔗 Cómo conectar la publicidad con esta página

Cuando ya tengas el link de tu página (ejemplo: `neuroeduca.com.gt/poder-invisible-mama/`):

1. **En tu publicidad de Meta Ads (Facebook/Instagram):**
   - En el campo "URL de destino" pones tu link
   
2. **En tus historias y posts orgánicos:**
   - Usa el sticker de "Link" en Stories
   - En posts, pon "Link en bio" y actualiza el link de tu bio
   
3. **En tu bio de Instagram:**
   - Usa Linktree o pon directamente el link
